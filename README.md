# Foundations of AI (CS 329) – Group Project

This repository contains a **group course project** completed as part of  
**CS 329: Foundations of AI – Multiagent Systems** at IIT Gandhinagar.

## Project Overview
This project explores concepts from multiagent systems through the design
and implementation of interacting agents in a simulated environment.

## My Contribution
As a member of the group, my contributions included:
- Implementing agent logic and behavior
- Assisting in debugging and experimentation
- Contributing to analysis and report preparation

## Repository Structure
- `agents/` – agent implementations
- `scripts/` – experiment and helper scripts
- `result/` – outputs and results
- `index.html`, `renderer.js`, `main.js` – visualization components
- `report.tex` – project report
